Tessellation Shaders (OpenGL 4+)

3 components: Tessellation Control Shader, Tessellator, Tessellation Evaluation Shader.

Tessellation Control Shader

Sends control points to Tessellation Evaluation Shader. Sends tessellation levels to Tessellator.

Tessellation Evaluation Shader

Applies control points to grid. Sends out resulting vertices. Does some configuration of the tessellator.

Tessellator

Recursively generates grid of triangles. Sends grid to Tessellation Evaluation Shader. Not programmable ("fixed function")
